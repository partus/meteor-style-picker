stylesSelection
===============

Dynamically change bootstrap them on the client (thembs from bootswatch.com).

Usage
-----

```
Session.set('bsStyle', 'name of the style')
```
Currently supported style names
-------------------------------

* ameila
* cerulean
* cosmo
* cyborg
* journal
* lumen
* slate
* spacelab


Installation
------------

```
mrt add style-picker
```
